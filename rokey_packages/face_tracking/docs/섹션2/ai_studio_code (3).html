기존 접근법의 한계와 해결책

❌ 기존 접근법: MoveIt IK 기반 제어

• 계산 지연(~50ms), 복잡한 경로, 불필요한 움직임

⭕ 우리의 해결책: Joint-space 직접 제어

• 계산 지연(<1ms), 단순 각도 매핑, 2-DOF 제어

❌ 기존 접근법: CPU 기반 추론

• YOLO 추론: ~50ms, 실시간 처리 불가

⭕ 우리의 해결책: GPU + TensorRT 추론

• YOLO 추론: ~5ms, 50Hz+ 처리 가능
