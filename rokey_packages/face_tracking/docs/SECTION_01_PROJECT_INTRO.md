##  1.1 프로젝트 개요

### 프로젝트명
> ** 로봇팔 초병 시스템 (Robot Arm Sentry System)**

### 한 줄 요약
> **"AI 기반 자동화 초병 시스템으로 24시간 무인 경계 실현"**

### 핵심 컨셉
```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│    군 위병소                     로봇팔 초병                 │
│   ─────────────                  ─────────────                  │
│   • 24시간 3교대 인력            • 24시간 무인 운용             │
│   • 피로/집중력 저하             • 일관된 AI 판단               │
│   • 인적 오류 위험               • 실시간 상황 전파             │
│                                                                 │
│              ────────────────────────►                          │
│                     자동화 전환                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

##  1.2 WHY? 왜 만들었나

### 현재 위병소 초병의 문제점

| 문제 | 설명 | 심각도 |
|------|------|--------|
|  **피로 & 집중력 저하** | 장시간 경계 근무로 인한 판단력 감소 |  높음 |
|  **인적 오류 위험** | 피아식별 실수, 암구호 확인 누락 가능성 |  높음 |
|  **야간/악천후 취약** | 시야 제한, 반응 속도 저하 |  중간 |
|  **인력 부족** | 24시간 3교대 인력 운용 부담 |  중간 |

### 문제점 시각화

```
┌─────────────────────────────────────────────────────────────────┐
│                    현행 위병소 초병 시스템                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   [야간 2시]                                                    │
│       │                                                         │
│       ▼                                                         │
│    초병 졸음 ─────►  침입자 접근 ─────►  감지 실패        │
│                                                                 │
│   [피아식별 상황]                                                │
│       │                                                         │
│       ▼                                                         │
│    피로한 초병 ───►  판단 실수 ─────►  아군 오인/적군 통과 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

##  1.3 WHAT? 무엇을 만들었나

### 로봇팔 초병 시스템 구성

| 구성 요소 | 기능 | 담당 |
|----------|------|------|
|  **로봇팔** | Doosan M0609 - 비비탄 권총 파지 & 조준 | 성우 |
|  **헤드샷 트래킹** | 실시간 얼굴 추적 (YOLOv8 + EKF) | 태슬라 |
|  **피아식별** | 아군/적군 분류 (Image Classification) | 경훈 |
|  **음성 암구호** | 암구호 확인 시스템 (Whisper STT + gTTS) | 지원 |
|  **웹 UI** | 실시간 상황 전파 & 알림 | TBD |
|  **아두이노** | 레이저 포인터, 차단봉 제어 등 | TBD |

##  1.6 기대 효과

### 정량적 효과

| 항목 | 현재 (인간) | 목표 (로봇) | 개선율 |
|------|-------------|-------------|--------|
| 연속 운용 시간 | 2~4시간 | 24시간 | +500% |
| 피아식별 정확도 | ~90% | >95% | +5%p |
| 반응 속도 | 0.5~1초 | <0.1초 | 10x |
| 운용 인력 | 3명/1개소 | 1명/다개소 | -66% |

### 정성적 효과

-  **보안 강화**: 24시간 무피로 경계
-  **인력 효율**: 병력 운용 최적화
-  **기록 자동화**: 모든 상황 자동 로깅
-  **확장성**: 다개소 동시 운용 가능
