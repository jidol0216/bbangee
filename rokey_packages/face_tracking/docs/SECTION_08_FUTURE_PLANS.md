### 기술 스택 (실제 구현)

| 구분 | 기술 | 설명 |
|------|------|------|
| **Frontend** | React + TypeScript | Vite 기반 SPA |
| **Backend** | FastAPI | Python REST API |
| **Database** | SQLite | SQLAlchemy ORM |
| **통신** | Axios | REST API 호출 |
| **하드웨어** | ESP32 | 서보/레이저 제어 |
| **TTS** | ClassifyVoice | 피아식별 음성 안내 🆕 |

### DB 구조

**People 테이블:**
| 컨럼 | 타입 | 설명 |
|--------|------|------|
| id | int | PK |
| military_serial | str | 군번 |
| name | str | 이름 |
| department | str | 소속 |
| rank | str | 계급 |
| picture | blob | 얼굴 사진 (Base64) |
| created_at | datetime | 등록일 |

**AccessLog 테이블:**
| 컨럼 | 타입 | 설명 |
|--------|------|------|
| id | int | PK |
| military_serial | FK | 군번 (People 참조) |
| in_time | datetime | 입실 시간 |
| out_time | datetime | 퇴실 시간 (nullable) |

### API 엔드포인트

| 메소드 | 엔드포인트 | 설명 |
|--------|-----------|------|
| GET | `/people/` | 전체 인원 조회 |
| GET | `/people/search/{serial}` | 군번 검색 |
| POST | `/people/register` | 인원 등록 (사진 포함) |
| DELETE | `/people/{serial}` | 인원 삭제 |
| GET | `/access/logs/{serial}` | 출입 이력 조회 |
| POST | `/access/{serial}/entry` | CHECK-IN |
| POST | `/access/{serial}/exit` | CHECK-OUT |
| POST | `/device/servo` | 서보 제어 |
| POST | `/device/laser` | 레이저 제어 |
| POST | `/voice/classify` | 피아식별 음성 안내 (TTS) 🆕 |
